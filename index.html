Python Australia Community

Python Australia Community is a community-run organisation dedicated to bringing together Python enthusiasts from across the country. We aim to connect Python-appreciators with local meetups, events, and initiatives to help grow and connect the Python community in Australia.

Join the Python Community Discord!

Connect with Python enthusiasts from all over Australia. Our Discord server is the perfect place to chat, get notified about events in your area, and discover new ways to get involved in the community. Please remember to abide by our Code of Conduct (courtesy of the PSF).

Local Python Meetups

SydPy (Sydney Python Meetup)

SydPy Meetup runs monthly in the Sydney CBD, follow our Luma calendar for events alerts.

We welcome Python-appreciators of all skill levels from all sectors. From students, to pro-devs, data scientists to UX engineers, we'd love to have you join us!

Frequency: Monthly
Location: Sydney, Central
