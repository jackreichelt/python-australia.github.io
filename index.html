Python Australia

Python Australia is a community-run organisation dedicated to bringing together Python enthusiasts from across the country. We aim to connect Python-appreciators with local meetups, events, and initiatives to help grow and connect the Python community in Australia.

Join the Python Australia Community Discord!

Connect with Python enthusiasts from all over Australia. Our Discord server is the perfect place to chat, get notified about events in your area, and discover new ways to get involved in the community. Please remember to abide by our Code of Conduct (courtesy of the PSF).

Local Python Meetups

SydPy (Sydney Python Meetup)

SydPy Meetup runs monthly in the Sydney CBD, follow our Luma calendar for events alerts.

We welcome Python-appreciators of all skill levels from all sectors. From students, to pro-devs, data scientists to UX engineers, we'd love to have you join us!

Frequency: Monthly
Location: Sydney, Central

Python WA

Python WA meets every month for talks and social events. All Python fans welcome!

Frequency: Monthly - First Thursday
Location: Perth, Western Australia

Canberra Python Meetup

Canberra Python Meetup runs monthly and welcomes Pythonistas of all skill levels. Join us for talks, workshops, and networking in the heart of Canberra.

Frequency: Monthly
Location: Canberra, ANU

Adelaide Python Meetup

Python Adelaide is a group for anybody interested in the Python Programming Language, who happen to be geographically located in close proximity to Adelaide.

Frequency: Monthly
Location: Adelaide

MelbPy (Melbourne Python Meetup)

The Melbourne Python User Group convenes monthly, bringing together Python enthusiasts and creators. We delve into Python and its associated open-source tools.

Frequency: Monthly
Location: Melbourne
